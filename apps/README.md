# Cross-platform Apps

This directory contains cross-platform apps written against package interfaces
instead of one hardware platform.


| App | Purpose |
| --- | --- |
| `glib/unit-test` | Runs the shared `glib.std` compatibility test suite through the selected platform runtime. |
| `zux/button-ledstrip` | Runs a spec-driven Zux button/LED-strip app and its user-story tests through the selected platform runtime. |
